# <center><font face="楷体" size=6>基本数据类型和String类型的相互转换</font></center>
## **一.基本数据类型转换成字符串**
1. 直接加上双引号` "" `就可以了
   ```
   int n1 = 1;
   String s1 = n1 + " ";
   ```
## **二.字符串转换成基本类型**
+ 使用基本数据类型对应的包装类的相应的方法
    ```
    String s5 = "123";
    int num1 = Integer.parseInt(s5);
    double num2= Double.parseDouble(s5);
    ```
