## **四.赋值运算符**
1. 形如 += -= %= 还有直接=
2. 赋值运算的特点：
    + 运算顺序是从右往左
    + 运算符的左边只能是变量，右边随意 可以是变量，表达式，常量值
    + 复合运算符(+=之类的)会强制类型转换
      ```
      byte b = 2;
      b += 3;//等价b = (byte)(b + 3)
      b++;// b = (b + 1)
      ```
## **五.三元运算符**
1. 基本语法： 条件表达式 ？ 表达式1 ： 表达式2;
   + 运算规则:
     + 条件表达式是true ，运算后的结果是表达式1;
     + 条件表达式是false ，运算后的结果是表达式2; 
2. + 表达式1和表达式2要为可以赋给接收变量的数据类型（或者可以自动转换）
        ```
        int a = 3;
        int b = 8;
        int c = a > b ? 1.1 : 3.4 //出错，double型无法赋值给int
        ```
        修改方法
        ```
        int c = a > b ?(int) 1.1 : (int) 3.4
        ```
