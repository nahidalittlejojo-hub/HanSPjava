## 一.Hello world
1. 
    ```
      public class Hello_world_main
      {
      public static void main(String[] args)
      {
        System.out.println("Hello World");
       }
       } 
      //这是java的快速入门
      //1.pulic class  Hello_world_main表示 Hello_world_main是一个类，是一个pubilc公有的类
      //2. Hello_world_main{ }表示一个类的开始和结束
      //3.public static void main(String[] args)表示一个主方法，即程序的入口
      //4.main () { }表示方法的开始和结束
      //5.System.out.println("Hello world");表示输出括号内的内容到屏幕上
     ```
2. 执行流程分析：
      .java文件（源文件）编译生成 .class文件（字节码文件）  运行出结果
## 二.JAVA开发注意事项和细节说明
1. + JAVA源文件以 .java为拓展名。源文件的基本组成部分是类(class)
    + java应用程序的执行入口是main()方法。他有固定的书写格式！
    + java严格区分大小写。每一个语句都以";"结束
    + 一个源文件中最多只能有一个public类。其他类的个数不做限制  每一个类编译以后都会有一个class文件
    + 如果源文件包含一个public类，则文件名必须按该类名命名!!
    + 也可以将main方法卸载非public类中，然后运行指定的非public类，这样入口方法就是非public的main方法。以非public的类Dog为例
       ```
       class Dog
       {
        public static void main(String[] args)
        {
            System.out.println("hello little gogo");
        }
       }
       ```
       编译后会生成Dog.class文件，使用指令java Dog指令将Dog类导入JVM运行
