## 一.转义字符
1. java常用的转义字符(便于输出某个字符在屏幕上)
+ \t：一个制表位，实现对齐的功能 光标位置往后跳8个字符位
+ \n：换行符
+ `\\`：一个\
+ `\"`：一个"
+ `\'`：一个'
+ \r：一个回车 光标回到行首
## 二.注释 
1. +  单行注释 `//`
+ 多行注释 `/* */`  不允许多行注释的嵌套！
## 三.Java代码规范
+ 类，方法的注释，要以Javadoc的方式来写
+ 非Java Doc的注释，往往是给代码的维护者看的，着重告诉读者为什么这么写，如何修改，注意什么问题
+ 使用tab操作，实现缩进，默认整体向右边移动，用shift+tab整体左移
+ 运算符含之间多打点空格
+ 源文件用UTF-8方式编码
## 四.DOS的基本原理
1. DOS 即磁盘管理系统 进行接受指令，解析指令，执行指令的操作
2. +  相对路径：从当前目录开始定位，形成的一个路径
   +  绝对路径：从根目录开始定位，形成的路径
3. 基本指令
  + 查看当前目录有什么  dir  dir d:\abc2\test200
  + 切换到其他的盘下面： 切换到C盘下 cd /D c:
  + 钱换到当前盘的其他目录  ..\表示上一级目录
    从D:\abc\test100切换到\abc2\test200: cd ../../abc2/test200
