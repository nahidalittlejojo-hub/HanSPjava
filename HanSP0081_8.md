## **一.标志符的命名规则和规范**
1. 标识符概念：你能起名的地方
2. 命名规则：
    + 可以由英文字母大小写，数字，_和$组成
    + 数字不可以开头
    + 不可使用已经存在的关键字和保留字(现有的java版本未使用但是优厚的版本可能会作为关键字使用)，但可以包含关键字和保留字
    + 严格区分大小写，没有长度限制
    + 不能包含空格
3. 命名规范
   + 包名: 多单词组成时所有字母小写
   + 类名: 多单词组成时候，所有单词的首字母大写
   + 变量名,方法名：多单词组成时，第一个单词首字母小写，第二个单词开始每个单词首字母大写
   + 常量名: 所有字母都大写。多单词时每个单词用下划线连接
   + 其他规则，未完待续
## **二.键盘输入语句**
1. 需要一个扫描器(对象),就是Scanner(Scanner是类)
2. 步骤：
   + 导入该类的所在包，java.util.*
   + 创建该类的对象（变量声明）
   + 调用里面的功能
3. 代码演示
   ```
   import java.util.Scanner;
   public class Input
   {
    public static void main(String[] args)
    {
        Scanner myScanner = new Scanner(System.in);
        System.out.println("请输入name:");
        String name =myScanner.next();//输入默认标记为字符串
        System.out.println("请输入age:");
        int age =myScanner.nextInt();//输入下一个标记为int
        System.out.println("请输入salary:");
        double salary = myScanner.nextDouble();//输入下一个标记为double

    }
   }
   ```
## **三.二进制**
1. + java没有无符号数
   + 在计算机运算的时候都是以补码的方式来运算的
2. |位运算符|含义|运算规则|
   |:------:|:--:|:---:|
   |&       |按位与|同1为1|
   |&#124;|按位或|有1为1|
   |^|按位异或|两个不同就是1|
   |~|按位取反|取反|
   + 注意：位运算是按照补码运算的！输出的结果是输出补码转换成原码后对应的数字！
3. |位运算符|含义|运算规则|
   |:------:|:--:|:---:|
   |>>|算术右移|低位溢出，符号位不变，用符号位补溢出的高位|
   |<<|算术左移|符号位不变，低位补0|
   |>>>|逻辑右移|低位溢出，高位补0|
